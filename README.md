
# ecan

The goal of ecan is to support ecological analysis.

## Installation

``` r
  # development
  # install.packages("devtools")
devtools::install_github("matutosi/ecan")
```

## Example

``` r
library(ecan)
library(tidyverse)
library(magrittr)
```

## Citation

Toshikazu Matsumura (2022) Ecological analysis tools with R.
<https://github.com/matutosi/ecan/>.
